# Shop2Game — واجهة شحن جواهر (Flask)

واجهة شحن جواهر لفريفاير مع خادم Flask بسيط.

التشغيل محليًا:

1. إنشاء بيئة افتراضية وتثبيت التبعيات:

```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install -r requirements.txt
```

2. تشغيل التطبيق:

```powershell
python app.py
```

3. افتح المتصفح على `http://127.0.0.1:5000`.
